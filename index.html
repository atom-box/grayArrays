Filter

General: FILTER has the same syntax as EVERY and SOME.

Definition: filter takes three arguments. It is common to write these as a tri-input function on the previous line and insert that function as a single argument into FILTER. If you build it they will come; the three args get an auto-feed, generated by the FILTER, which, after all, is a loop-like method. When invoked, FILTER will run a loop, and with each turn of the wheel it will access (value, index, array) based on the array that called it.
